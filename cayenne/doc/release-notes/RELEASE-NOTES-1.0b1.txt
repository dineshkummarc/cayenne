           Cayenne Release Notes 1.0b1 (Beta 1)
                 Date: [unreleased]
         ============================================

For the latest information go to the project web site at 
http://objectstyle.org/cayenne/



Changes in this release:
============================================
1. Added ResultIterator - now it is possible to retrieve data rows one at a time.

2. Added public method DataContext.objectFromDataRow (rather exposed 
another private method).

3. Term "Snapshot" is changed to "DataRow" in most places.
   - OperationObserver and its implementations: nextSnapshots became nextDataRows
   
4. Removed public DataNode.snapshotsFromResultSet refactored into 
DefaultResultIterator

5. It is no longer recommended to install Cayenne in JVM extension 
directory due to resource loation issues.

6. Split OperationObserver into OperationObserver and OperationHints for clearer API.

7. Renamed SelectOperationObserver to SelectObserver for consistency reasons.
