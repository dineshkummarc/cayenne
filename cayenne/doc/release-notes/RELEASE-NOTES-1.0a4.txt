           Cayenne Release Notes 1.0a4 (Alpha 4)
                 Date: October 21, 2002
         ============================================

For the latest information go to the project web site at 
http://objectstyle.org/cayenne/



Changes in this release:
============================================
1. Removed all methods that were deprecated prior to Aplpha 3.

2. Synchronized internal DataContext cache access.

3. Added "unregisterObject" method to DataContext 

4. Made DataContext and DataObjects serializable - a must for web 
applications.

5. Added "paged" queries support

6. Cleaned up tutorials, fixed tutorial bugs and inconsistencies.

7. Refactored DataContext, created helper classes, removed unused 
methods

8. Refactored util.Preferences into gui.ModelerPreferences and
project.CayenneUserDir
