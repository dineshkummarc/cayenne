This directory contains 3rd party Java libraries. 

These should NOT be required to run cayenne. 
Normally these JAR's and ZIP's are used during compilation and JUnit test runs.

1. junit.jar		- JUnit unit testing package
2. JavaCC.zip		- Java Compiler Compiler, parser generator used to create 
			  WO-compatibility parser classes	  

These may be required in runtime:

1. velocity-*.jar 	- Jakarta Velocity template engine
